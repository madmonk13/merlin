MERLIN
THE ELECTRONIC WIZARD

HOW TO PLAY MERLIN

Game selection works just like the original handheld:

  1. Press NEW GAME
  2. The first four lights begin blinking
  3. Press a numbered button to choose:
  • 1 Echo — memory sequence
  • 2 Tic-Tac-Toe — 3×3 grid
  • 3 Music Machine — each button a note
  • 4 Magic Square — turn off all lights

Other controls

  • SAME GAME — restart the current game
  • HIT ME — replay your tune (Music) or the sequence (Echo)
  • COMP TURN — let Merlin move first (Tic-Tac-Toe)
